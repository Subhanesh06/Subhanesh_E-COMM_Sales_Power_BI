# Subhanesh_E-COMM_Sales_Power_BI

# 🛍️ Subhanesh E-Commerce Sales Dashboard 🚧

An interactive business intelligence dashboard designed to analyze e-commerce sales performance, track profitability, and uncover key customer buying patterns.

---

# 📌 Overview

Subhanesh E-Commerce Sales Dashboard is a data analytics project designed to simulate real-world retail sales tracking and operational performance evaluation.

It includes:

* 📈 Executive KPI tracking (Sales, Profit, Quantity, AOV)
* 🗺️ Regional breakdown of sales volume across top states
* 💳 Analysis of payment modes and transactional preferences
* 📊 Product category and sub-category profitability tracking

The system provides a clear, actionable data view wrapped in a professional green gradient user interface, optimized for business decision-making.

---

# ✨ Features

📈 Dynamic KPI Tracking (438K Total Sales, 37K Profit, 5.6K Quantity)

📉 Monthly Profit-Loss Trend Analysis (Identifying seasonal peaks & drops)

🗺️ Regional Market Penetration Analysis (Top States tracking)

📊 Product Category Breakdown (Clothing, Electronics, Furniture share)

💳 Payment Mode Distribution Analysis (COD vs. UPI vs. Cards)

👥 Top Customers Leaderboard for loyalty tracking

🎛️ Interactive Filtering via Quarterly Slicers (Qtr 1 - Qtr 4)

🎨 High-contrast, custom green gradient dashboard layout
